Настройка токенов доступа
Создайте файл .env в корне проекта и добавьте токены:

env
# Токен VK API
# Получить: https://vk.com/apps?act=manage → Создать Standalone приложение → Скопировать токен
VK_ACCESS_TOKEN=ваш_токен_vk_здесь

# Токен Яндекс.Диска  
# Получить: https://yandex.ru/dev/disk/poligon/ → Получить OAuth-токен
YD_ACCESS_TOKEN=ваш_токен_яндекс_диска_здесь
